How I did it

Posted yesterday

This blog is written using HTML and CSS. I feel like I'm getting the hang of it.

HTML tags are just written like:

<h1>Hello World</h1>

But there are a lot of tags you need to know!

CSS is a lot more complicated, but there are two parts:

Selectors
let you say which tags you want to style
Properties
let you say what style you want to make those tags

But there are even more of those than there are HTML tags! But don't worry, I'm sure I'll learn enough of them soon!
